# WPF.App.CommunityToolkitMVVM
## WPF 
- CommunityToolkit.Mvvm을 이용한 MVVM 구조의 WPF App.
- ObservableRecipient의 Messenger를 이용한 ViewModel간의 통신.
- CustomControl 작성 예시 (RadioButton 스타일 기반의 토글버튼)
## Interface를 이용한 모듈화 예제 [AExcel (using OpenXML)]
- 오픈소스 OpenXML을 Interface를 통하여 선행 지식 없이 좀더 쉽게 엑셀 문서를 생성할 수 있는 라이브러리.
- 엑셀파일 Create/Open/Save 기능과 간단한 기능 구현 (Cell에 문자 입력)
## SimpleTestAutomation 예제 [Python]
- SimpleTestAutomation은 Python 기반의 간단한 테스트 자동화 엔진입니다.
YAML 설정 파일을 기반으로 테스트를 실행하고 결과를 생성합니다.
본 WPF 애플리케이션은 해당 엔진을 독립 실행 파일(EXE) 형태로 실행하며,
외부 프로세스의 표준 출력(stdout)을 수신하여 결과를 UI에 시각화합니다.
